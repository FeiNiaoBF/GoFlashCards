# GoFlashCards

一个用 Go 和 Flutter 实现的类 Anki 项目

## 介绍

**GFC**(GoFlashCards) ,是我第一个用 Go 写的一个小项目，
可以让我轻松制作类 Anki 卡片并用来存储和测试自己的知识

## 技术栈

- Go 建立后端
  - echo
  - templ
  - sqlc
- Flutter 建立前端
- PostgreSQL 数据库
